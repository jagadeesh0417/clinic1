# Public assets

Drop static assets here (favicon, og-image.png, brand imagery, clinic photos).
Recommended:
- favicon.ico
- og-image.png (1200×630) for social sharing
- /clinics/*.webp for clinic galleries (or serve via Cloudinary)

Brochure-style luxury clinic photography and beauty imagery referenced in the
design can be hosted on Cloudinary and referenced with next/image
(the cloudinary remote pattern is already whitelisted in next.config.mjs).
